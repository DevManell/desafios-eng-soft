# Site Engenharia de Software

Projeto desenvolvido para uma atividade acadêmica com foco em:

- HTML semântico
- Acessibilidade Web
- Estruturação correta de páginas
- Melhoria da navegação e leitura

## Objetivo

O objetivo do projeto foi corrigir e melhorar uma página HTML utilizando boas práticas de desenvolvimento web e acessibilidade.

## Melhorias Implementadas

- Uso de tags semânticas:
  - `<header>`
  - `<main>`
  - `<section>`
  - `<article>`
  - `<footer>`
  - `<nav>`

- Hierarquia correta de títulos
- Implementação de modo escuro
- Melhor contraste visual
- Link “Pular para conteúdo”
- Imagens com atributo `alt`
- Navegação acessível
- Estrutura responsiva

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
